## build locally

```sh
# install bundler
sudo gem update
sudo gem install bundler

# install (in folder)
bundle install

# test locally
bundle exec jekyll serve

# build locally
bundle exec jekyll build

# build dat
asset/config/dat/build-dat.sh
```
